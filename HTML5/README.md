# Web Accessibility Site Developed in MadCap Flare
This website serves to showcase my understanding of disability and web accessibility while demonstrating my knowledge of MadCap Flare. The first section, titled "Web Accessibility," gives a summary of disability, explains multiple approaches to web accessibility, and highlights the importance of universal design. This section was developed by me individually using readings and classwork done in my English 381 course. The second section, titled "Video Accessibility: Closed Captions," defines closed captions, articulates their importance to accessibility, offers a brief history of closed captions, and gives step-by-step instructions for implementing closed captions into videos on Microsoft's Clipchamp. This section was created collobaratively within a small team of classmates.

## Objectives
- Demonstrate my understanding of web accessibility, a crucial topic for any web developer or content creator, to potential employers.
- Showcase my ability to utilize MadCap Flare to create web content, increasing the tools I am proficient with.
